СТРОГО соблюдаем формат входных и выходных данных.
Все массивы должны вводиться с клавиатуры в формате: e0 e1 ... eN (Каждый елемент через пробел)
Для каждой задачи указано что конкретно ожидается в выводе.
В выводе НЕ ДОЛЖНО быть текста помимо указанного для каждой задачи.
Не забываем про перенос строки (\n) в printf, что бы вывод не смешался.
Разделять вывод для разных задач можно пустыми строками, либо простым текстом типа:
	Task #1 output: <text>

Пожалуйста, присылайте код одним файлом

Массивы выводятся в формате: e0 e1 e2 ... eN

Двумерные массивы в формате: e00 e01 ... e0N
                             e10 e11 ... e1N
                             ...
                             eM0 eM1 ... eMN


1. Ввести целочисленный массив, состоящий из 14 элементов. Вычислить
количество и сумму четных по значению положительных элементов.
Вывести через пробел: количество, сумму

2. Ввести целочисленный массив, состоящий из 17 элементов. Заменить
элементы кратные трем на сумму нечетных по значению элементов.
Вывести через пробел элементы преобразованного массива

3. Ввести массив, состоящий из 14 элементов действительного типа. Распо-
ложить элементы c 1 по 7 по возрастанию, а с 8 по 14 - в порядке убывания.
Вывести через пробел элементы преобразованного массива

4. Ввести массив, состоящий из 12 элементов действительного типа. Опре-
делить количество элементов, стоящих между максимальным и минимальным
элементами. Если несколько минимальных - взять тот, у которого наименьший индекс.
Если несколько максимальных - взять тот, у которого наибольший индекс.
В выводе одно число: количество элементов

5. Ввести массив, состоящий из 15 элементов целого типа. Определить
количество отрицательных, произведение положительных и количество
нулевых элементов.
Вывести через пробел: кол-во отрицательных, произведение, кол-во нулевых

6. Ввести массив, состоящий из 12 элементов действительного типа. Опре-
делить границы интервала, в котором находятся значения элементов
массива.
В выводе два числа через пробел: нижняя и верхняя границы

7. Дан массив - 19 элементов целого типа. Найти сумму элементов, распо-
ложенных до первого отрицательного элемента. Если отрицательных
элементов нет, то вывести -1.
В выводе одно число: сумма (или -1)

8. Ввести матрицу, состоящую из 4x4 элементов целого типа. Транспонировать заданную матрицу.
Вывести полученную матрицу

9. Ввести матрицу, состоящую из 4x4 элементов целого типа. Поменять местами диагонали матрицы.
Вывести полученную матрицу

10. Ввести матрицу, состоящую из 5x5 элементов целого типа. 
Найти сумму элементов матрицы, лежащих внутри прямоугольника,
ограниченного максимальным и минимальным элементами маирицы.
Если несколько минимальных - взять тот элемент, который ближе к левому верхнему углу матрицы.
Если несколько максимальных - взять тот элемент, который ближе к правому нижнему углу матрицы.
В выводе одно число: сумма
